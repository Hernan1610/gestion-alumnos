# Sistema de Gestión de Alumnos

Aplicación web desarrollada con JavaScript para gestionar alumnos y pagos.

## Funcionalidades

- Agregar alumnos
- Registrar pagos
- Eliminar alumnos
- Persistencia de datos con LocalStorage

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript
